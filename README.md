fis-prepackager-pai-sf
==========================
派qb页接入sf的插件

This plugin is to make vars compatible both in https and superframe for ik.



> 安装

```javascript
    npm install -g fis-prepackager-pai-sf
```

> fisp fis-conf.js中配置

```javascript
    fis.config.set('modules.prepackager', 'widget-inline,pai-sf');
```

> fis3 fis-conf.js中配置

```javascript
    fis.plugin('iknow-sf')
```
